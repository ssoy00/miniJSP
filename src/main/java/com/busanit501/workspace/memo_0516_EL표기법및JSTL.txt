JSP - EL (Expression Language)
jsp 코드에서 출력 용으로 사용하는 표현식이다.

사용법은 ${} , 중괄호에 서버에서 지정한 키의 이름을
이용해서 가져오기.
list 10개의 더미 데이터 가지고 있다
예) ${list}
예2) ${list[0].tno}
예3) ${list[0].title}
예4) ${list[0].dueDate}

JSP, 안에서, 반복문, 조건문, 출력, 변수 사용하는 부분,
좀 더 쉽게 작성하고 싶어요. 그래서, 도구를 이용하는데.
by java , 이용해서.

사용법
1) 반복문
문법 정의, <c:forEach>  태그 이용하기.
속성,
var(하나 꺼내서 사용할 요소의 변수명) ,
 items(목록 리스트 ),
 begin/end(인덱스에서 순번 정하기.)
예)
<ul>
  <c:forEach var="dto" items="${list}">
    <li>
      ${dto}
    </li>
  </c:forEach>
</ul>
